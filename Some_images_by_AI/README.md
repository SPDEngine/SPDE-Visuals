# Image Collection

This folder contains a variety of images, most of which were generated using MidJourney (denoted by the prefix **MJ** or my username on discord **chenle02**). The file names are based on the prompts used for generating the images. One image, `Nano_Banana_Pro_Classical_to_Singular_SPDEs_regularity_barrier.png`, was generated by **Gemini 3** using **Nano Banana Pro**. Below is an organized breakdown of the images by their themes and purposes. These images were used in some of my past talks.
